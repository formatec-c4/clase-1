# vet-firulais-turnos

Mock simple de turnos online para la `Clinica Veterinaria Firulais`.

## Stack

- Python 3
- `http.server`
- Bootstrap 5 por CDN

## Ejecutar

```bash
python3 app.py
```

Luego abrir `http://127.0.0.1:8000`.
